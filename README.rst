Pyveplot
========


`Hiveplots <http://www.hiveplot.com/>`_ are a nice way of visualizing 
complex networks.

This library uses `svgwrite <http://svgwrite.readthedocs.org/en/latest/classes/shapes.html>`_ to 
programbatically create hiveplots.
